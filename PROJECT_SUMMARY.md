## 🛠 技术栈

- **后端**: Go + Gin + XORM
- **前端**: UniApp + Vue.js
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **API**: RESTful API设计

## 🎯 部署建议

### 开发环境
```bash
# 启动后端
cd backend
go run main.go

# 前端开发
cd frontend
# 使用HBuilderX或uni-app CLI开发
```

### 生产环境
1. 配置PostgreSQL数据库
2. 修改config/database.go中的数据库连接
3. 编译Go程序：`go build -o app main.go`
4. 部署UniApp到各个平台
